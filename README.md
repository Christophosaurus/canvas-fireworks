# Starter project for canvas development using typescript and scss

To run the project in development

```zsh
npm run dev
```
Runs a webpack dev server on `localhost:42069`


To build the project

```zsh
npm run build
```

For some helpful dev stuff

```zsh
# prettier
npm run format

# es-lint
npm run lint

```
